404 錯誤頁面

404 Not Found
何時出現
URL 路徑不存在、拼字錯誤、文章永久刪除/下架、時效性專區移除
Server 找不到對應路由 → 直接回 HTTP 404
核心元件
搜尋框（P0）+ 回首頁按鈕（P1）
部署方式
應用程式內部渲染（維持原 URL）
Header
Sticky，含完整導航
標題文案
「很抱歉，頁面無法使用或被移除，試試別的吧?」
Footer
隨頁面流動（Non-sticky）
開發注意事項
  • 1404 必須在原 URL 回傳 HTTP 404，禁止 redirect 到 /404 路徑
  • 2前端禁止主動 redirect 至 404，錯誤頁由 Server 被動產生
  • 3無效分頁參數（如 ?page=9999）用 302 導回第一頁，禁止 301
  • 4GA4 追蹤：統一 Page Title「404 找不到網頁 | TVBS 新聞網」，不加 UTM / from 參數
  • 5Slug 變更時 CMS 須自動建立 301 Redirect 並同步更新 canonical
  • 6局部元件失效（廣告、非核心 API）用 Error Boundary 隱藏，核心內容在就回 200
3/26 Refinement — 決議移除項目
  • 自訂 500 頁面 — 改採系統 / CDN / Nginx 預設錯誤回應（必須回傳 HTTP 500/502/504）

Cookie Banner 隱私權置底橫幅

✓ 點擊「我同意」
橫幅立即隱藏（display: none）
寫入 Cookie
Key：tvbscookie
Value：{ status: 1, timestamp: ISO-8601 }
Domain：tvbs.com.tw
Expires：寫入起算 365 天
✕ 點擊「X」關閉
橫幅立即隱藏（display: none）
不寫入任何 Cookie 或 localStorage
下次重新整理或進站，橫幅再次顯示
顯示邏輯
條件行為
Cookie 不存在顯示橫幅
Cookie 存在，timestamp 未逾 365 天不顯示
Cookie 存在，timestamp 已逾 365 天視為過期，重新顯示橫幅
使用者點 X（無 Cookie 寫入）下次進站重新顯示
注意事項
  • 1position: fixed; bottom: 0; width: 100%
  • 2z-index 建議 999，低於蓋板廣告（Overlay Ad）或 Modal
  • 3「隱私權政策」連結開新分頁 → account.tvbs.com.tw/policy/privacy
  • 4橫幅不應自動無互動消失；換裝置 / 瀏覽器 / 清資料均視為新訪客
  • 5MVP 不強制無障礙（無須 aria-label / 鍵盤完整操作）

搜尋結果頁 — 區塊與互動

搜尋框
  • 輸入值先 .trim()
  • trim 後空字串 → 不搜尋
  • Enter 或點搜尋 icon 觸發
  • 等待期間不顯示 Spinner / 骨架屏
搜尋摘要
  • 搜尋：{關鍵字} 結果共 {N} 筆
  • N = API total
  • 關鍵字超過 10 非空白字元截斷 + ...
結果列表
  • 初始 16 則 + 載入更多 16 則
  • 排序：updated_at DESC
  • MVP 僅搜尋文章
  • 影音新聞加 Play overlay
  • 點擊另開新分頁

⬆ Scroll to Top 行為

觸發時機：搜尋 API 成功回傳並完成渲染結果列表或 Empty State 後
捲動方式：scroll to Y=0，smooth 平滑捲動（與文章頁 §3.4 一致）
不觸發：「載入更多」僅追加結果時不觸發，避免打斷使用者閱讀位置
3/26 Refinement — 決議移除項目
  • 歷史搜尋功能（localStorage、標籤、X 單則刪除、清除全部）— 移至 Out of scope
  • Spinner / 骨架屏等待畫面 — 等待期間不顯示任何載入指示
  • 新增：結果渲染後 Scroll to Top（smooth），與文章頁一致

搜尋結果頁 — 狀態與降級處理

SEO

Title：搜尋結果：{關鍵字} - TVBS新聞網
Robots：noindex（不被搜尋引擎索引） | 不需要 OG Tags、不需要 JSON-LD、不設 Canonical

應用程式層狀態一覽
狀態觸發條件畫面行為
搜尋服務失敗OpenSearch 5xx / fetch timeout顯示「搜尋服務暫時無法使用」；搜尋框仍可操作
結果為空API 正常，結果 0 筆「找不到與『{關鍵字}』相關的結果，請嘗試其他關鍵字」
搜尋等待等待 API 回應不顯示 Spinner / 骨架屏；結果渲染後自動 scroll to top
載入更多失敗分頁 API 失敗 / timeout按鈕變「再試一次」；已載入結果不動
空白輸入.trim() 後為空字串不執行搜尋；維持當前狀態
局部失效不影響整頁 ｜ 可重試的錯誤保留使用者的關鍵字
